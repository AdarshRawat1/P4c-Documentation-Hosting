Welcome to p4 Compiler's documentation!
============================================

``p4c``  is a reference compiler for the P4 programming language. It supports both P4-14 and P4-16; you can find more information about P4 in our carefully currated Documentation.

   ``P4 Compiler's Documentation`` is a work in progress. This version contains mostly temporary placeholder functionalty.
